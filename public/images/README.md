# Image Directory Structure

This directory contains placeholder images for the AquariumChat application.

## Folder Structure

- `/hero/` - Hero section images for homepage
  - hero1.png, hero2.png

- `/community-chat/` - Community chat room images
  - community1.png, community2.png, etc.

- `/private-chat/` - Private chat session images
  - private1.png, private2.png

- `/profile/` - Profile page images
  - avatar-placeholder.png, badge-placeholder.png

- `/categories/` - Category images
  - `/fish/`
    - `/saltwater/` - Saltwater fish species images
    - `/freshwater/` - Freshwater fish species images
    - saltwater1.png, freshwater1.png
  - `/plants/` - Aquatic plant images
  - `/coral/` - Coral type images
