# Image Folder Structure

This document describes the folder structure for all placeholder images in the application.

## Homepage Cards
- `/images/home-cards/community-chat/` - Community Chat card on homepage
- `/images/home-cards/private-chat/` - Private Chat card on homepage  
- `/images/home-cards/ai-advisor/` - AI Advisor card on homepage

## Hero Images
- `/images/hero/hero1.png` - Main hero background image
- `/images/hero/hero2.png` - Secondary hero background image

## Community Chat Rooms
- `/images/community-chat-rooms/freshwater/` - Freshwater room
- `/images/community-chat-rooms/saltwater/` - Saltwater room
- `/images/community-chat-rooms/reef/` - Reef Systems room
- `/images/community-chat-rooms/community/` - Community Tank room
- `/images/community-chat-rooms/photos/` - Photos & Stories room

## Private Chat Rooms
- `/images/private-chat-rooms/saltwater/` - Saltwater private chat
- `/images/private-chat-rooms/freshwater/` - Freshwater private chat

## Profile
- `/images/profile/avatar/` - User avatar placeholder

## Categories
- `/images/categories/saltwater/` - Saltwater main category
- `/images/categories/freshwater/` - Freshwater main category
- `/images/categories/fish/` - Fish category images
- `/images/categories/coral/` - Coral category images
- `/images/categories/plants/` - Plant category images
- `/images/categories/chemistry.png` - Water chemistry
- `/images/categories/aquascape.png` - Aquascapes
- `/images/categories/photos.png` - Photos
- And many more subcategories (see categories.js)

## Usage
Replace placeholder.png files in each folder with your actual images. The app will automatically use them.
